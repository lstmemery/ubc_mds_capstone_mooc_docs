edXviz: Interactive Visualization of Student Engagement with edX MOOCs

8 Eiffel Tower Overview

8.1 Motivation

Since video and problem are two most basic components in edX MOOC courses, instructor can get a general picture about student engagement by knowing how many students watched/attemped each video/problem.

So we designed this Eiffel Tower engagement overview visulization to show the course structure vs.student engagement. Each bar is a course element (video, problem or module seperator), which is sorted based on their location in the whole course structure from the beginning to the end. The length of each bar indicates how many student engaged with this course element.

8.2 Purpose

This visulization aims to help instructor finish four types of tasks :

  • Obtain an intuitive sense about course structure. For example, instuctor can explore which module are excessive intensive containing too many elements.

  • See where did the students dropout.

  • Quickly locate to a specific video/problem by applying module filtering.

  • Explore pattern by applying demographic filtering. For example, instuctor can explore whether there is engagement difference between audit student and verified student.
